**修复与纠偏**:
1.  **严重张冠李戴修复**
    *   `polity_0092` **汉(551-552)**: 从错误的“成汉/四川重庆”修正为侯景之乱中侯景于建康建立的短期汉政权，现代近似改为南京及长江下游局部地区。

2.  **过度扩张口径收敛**
    *   西汉/新/东汉/西晋: 将“吉尔吉斯斯坦、塔吉克斯坦大部”等稳定领土口径改为“西域都护/长史府、塔里木盆地及中亚东缘绿洲影响区”，避免把阶段性都护或影响区等同现代国家全境。
    *   唐朝/武周: 将“蒙古国全境、俄罗斯远东、朝鲜半岛大部”等表述改为都护府、军镇、羁縻和阶段性控制口径，明确安东、安北、安西/北庭等机构不能直接等同现代国家全境。
    *   元朝: 删除“北达北冰洋”领土口径，并将高丽改为“半自治藩属”，避免把朝鲜半岛大部并入元朝直辖领土。
    *   明朝: 将奴儿干都司、黑龙江下游/库页岛及西南土司改为名义羁縻、朝贡卫所或阶段性控制口径。
    *   清朝: 保留外蒙古、外满洲、唐努乌梁海、外西北等极盛范围，但细化为巴尔喀什湖以东/以南、吉尔吉斯斯坦北部及东部、帕米尔争议/影响区等更精确表述。
3.  **局部现代地理规范化**
    *   前仇池、后仇池由历史地名“武都、阴平/陇西、汉中、天水”规范为甘肃东南、陕西西南、四川西北。
    *   东晋、刘宋、南齐、梁、陈涉及交州/越南的描述改为“越南北部及中北部/实际控制有时期差异”，避免将所有南朝阶段一概写成稳定控制越南中部。
4.  **缺失项备案**
    *   上一条 Batch 19 release note 提到的匈奴、突厥、回鹘、吐蕃、南诏、大理、高句丽、渤海并不存在于当前 `chinese_history_polities_master_v03.csv`，本次未能验证为已交付数据，列为后续补录 backlog。
    *   生成 pipeline 仍只加载中国 ADM1 边界；CSV 中蒙古国、俄罗斯、中亚等文本目前不会自动生成全球 polygon。真正的全球疆域展示仍需全球 ADM1 边界或历史 GIS 几何数据。
