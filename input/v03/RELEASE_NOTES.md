# v03 Release Notes

本文件记录 v03 数据集的数据口径、源表、生成表和公开产物变更。以后每次修改 `input/v03` 或重新生成 `public/data/v03` 时，都应在这里追加一条记录，说明变更原因、涉及文件、生成影响和校验结果。

## 2026-05-20 - 二次审计：名称层级与时间线关联修复

### 背景

继“仇池”聚合条目修正后，继续复核 v03 是否还有地名、族群、概念层与具体政权混用，以及历史事件、政权更迭、典故关联是否把未来政权或已灭亡政权提前/滞后挂载。

### 数据口径

- 不再把 `丁零` 作为 330-392 年连续年度政权展开；保留 `翟魏` 为 388-392 年丁零翟氏实际政权，并把“丁零”保留为别名/语境。
- 不再把 `周国` 作为前1046年的独立年度政权展开；该条更接近周人发祥地、王畿或宗庙概念，避免与 `西周` 混列。
- 将重复的 `段部` 并入 `段部鲜卑`，保留“段部”为别名，避免 310-339 年重复显示。
- 将 `秦国` 口径止于前221年，前221年转为 `秦朝`；前206年灭亡事件归入秦朝，不再生成“秦国亡于西楚/汉朝”。
- `新朝` 按公元9年王莽称帝建新口径开始，年度展开不再包含8年；`西汉` 年度止于8年，源表文案同步说明9年建新。

### 源数据变更

- `chinese_history_polities_master_v03.csv`
  - 删除 `polity_0001` / `丁零`、`polity_0058` / `周国`、`polity_0087` / `段部`。
  - 更新 `polity_0088` / `段部鲜卑` 的别名、消歧说明和灭亡方。
  - 更新 `polity_0112` / `翟魏` 的丁零别名和消歧说明。
  - 修正 `沈国` 亡于蔡国、战国 `韩国` 亡于秦国、战国 `魏国` 亡于秦国。
  - 修正 `秦国`、`新朝`、`西汉` 的起止口径与源表文案。
- `chinese_history_polities_yearly_v03.csv`
  - 删除丁零、周国、段部的年度行。
  - 删除新朝8年年度行。
  - 为秦国补齐前221年转入秦朝前的年度行。
  - 同步上述政权的源表字段并重新编号 `row_id`。
- `chinese_history_rulers_master_v03.csv`
  - 删除聚合性丁零条目的君主记录。
- `historical_anecdotes_v03.csv`
  - 将 `商鞅立木` 关联从秦朝改为秦国。

  - 修正楚汉、秦末和东汉末典故的政权关联，避免把西汉、曹魏、蜀汉提前挂到成立前。
- `chinese_history_unresolved_or_disputed_v03.csv`
  - 新增丁零聚合层不展开、段部重复合并、周国概念层不展开、秦国转秦朝的口径说明。

## 2026-05-20 - 仇池口径与三国时间线校正

### 背景

此前 `polity_0011` 使用聚合性的“仇池”条目，时间范围为 184-443 年。该条目把仇池地名、前仇池、后仇池等口径合并到一个年度政权中，导致 220-280 年三国时间轴里出现后续西晋、东晋十六国时代的仇池相关数据，且与 `前仇池`、`后仇池` 发生重叠。

### 数据口径

- 不再把聚合性“仇池”作为可展开的年度政权。
- 保留 `前仇池` 为 296-371 年氐族杨氏政权，灭于前秦。
- 保留 `后仇池` 为 385-443 年氐族杨氏复立政权，灭于北魏。
- “仇池”保留为 `前仇池`、`后仇池` 的别名和检索入口，而不是 184-443 年连续存在的独立政权。
- 三国相关时间线口径拆细为：
  - 221-263 年：三国鼎立与南北攻守。
  - 264-265 年：魏吴对峙与蜀汉覆亡后局势。
  - 266-280 年：西晋代魏与晋吴对峙。

### 源数据变更

- `chinese_history_polities_master_v03.csv`
  - 删除聚合条目 `polity_0011` / `仇池`。
  - 为 `polity_0018` / `前仇池` 增加别名 `前仇池国;仇池氐杨氏;仇池`。
  - 为 `polity_0043` / `后仇池` 增加别名 `后仇池国;仇池氐杨氏;仇池`。
  - 为前后仇池增加消歧说明，明确区别于仇池地名和彼此。
- `chinese_history_rulers_master_v03.csv`
  - 删除关联聚合条目 `polity_0011` 的君主记录。
- `chinese_history_polities_yearly_v03.csv`
  - 删除 `polity_0011` 的年度展开行。
  - 重新编号 `row_id`。
- `chinese_history_unresolved_or_disputed_v03.csv`
  - 增加“仇池聚合条目不展开”的前向和后向兼容说明。
- `historical_events_v03.csv`
  - 移除聚合性“仇池”建国/灭亡自动事件。
  - 保留 `前仇池`、`后仇池` 的建国和灭亡事件。
- `historical_activity_gaps_report_v03.csv`、`historical_event_coverage_report_v03.csv`、`chinese_history_validation_report_v03.csv`
  - 随源数据和事件重新生成。

### 生成逻辑变更

- `scripts/build_historical_events_v03.py`
  - 将 `three_kingdoms_balance` 从 221-280 年调整为 221-263 年。
  - 新增 `wei_wu_duality`，覆盖 264-265 年。
  - 新增 `jin_wu_duality`，覆盖 266-280 年。

### 公开产物变更

- 重新生成 `public/data/v03`。
- `alias_index.json` 中，“仇池”现在指向 `前仇池` 和 `后仇池`，不再指向聚合性年度政权。
- `contexts.json`、`events.json`、`polities.json`、`metadata.json`、`territories/*` 和年度 `years/*.json` 随源表同步更新。
- 关键年份校验：
