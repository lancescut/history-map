# China ADM1 Boundary Source

This directory stores the offline boundary source used by the v1 modern-admin
territory approximation pipeline.

- `china_adm1_geoboundaries_raw.geojson` is the pinned raw source downloaded
  from geoBoundaries `CHN ADM1 gbOpen`.
- `china_adm1_normalized.geojson` is generated by
  `scripts/prepare_admin_boundaries.py`.
- `admin_boundary_source_manifest.json` records source URL, license, hashes,
  feature count, and geometry quality checks.

These boundaries are modern administrative references used to make approximate
territory visualization legible. They are not historical borders.
